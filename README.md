# 富士急バス リニューアルデザイン（テスト公開）

https://bus.fujikyu.co.jp/ のリニューアル案として作成した、単一HTMLファイルのデザインモック。

## 公開

このリポジトリを GitHub Pages で公開すると、`index.html` がそのまま表示されます。

- ブランチ: `main`
- 公開パス: `/` (ルート)

## ファイル構成

```
index.html          — デザイン本体（Noto Sans JP + Inter / Tailwindなし、インラインCSS）
images/             — セクション背景写真
  hero_bus2.jpg     — ヒーロー（富士急バス車両）
  regions_bg.jpg    — 「どこへ行こう？」セクション
  services_bg.jpg   — 3 つのバス柱セクション
  news_bg.jpg       — ニュース + おすすめ
  tickets_bg.jpg    — おトクな乗車券
  safety_bg.jpg     — 安全への取り組み
```

## 写真クレジット

- Pexels (無料ライセンス / 商用利用可・帰属不要)
- 一部は手持ち画像（bus / 車両写真）
